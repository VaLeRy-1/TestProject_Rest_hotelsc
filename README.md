# Hotel API

RESTful API for managing hotels with features like search, filtering, and statistics.

## Technologies

- Java 21
- Spring Boot 3.2.3
- Spring Data JPA
- Liquibase
- H2 Database (with support for MySQL/PostgreSQL)
- MapStruct
- Lombok
- Swagger/OpenAPI 3
- Maven

## Prerequisites

- JDK 21
- Maven 3.8+
- Docker (optional, for MySQL/PostgreSQL)
